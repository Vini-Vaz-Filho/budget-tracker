# Poker Game Python Project

## Description
This is a Python-based Poker Card Game with a graphical user interface built using Tkinter. It allows users to:
- Play against the computer
- Receive random 5-card hands
- Reveal the computer's hand
- Automatically compare hands
- Keep score across rounds
- View game rules in a popup window
- Reset scores at any time

## Features
- Uses Object-Oriented Programming
- Uses Tkinter GUI
- Includes input validation / error handling
- Uses lists, dictionaries, loops, and functions
- Random card shuffling
- Hand ranking system
- Tie-breaker logic
- Score tracking system

## Hand Rankings
- Three of a Kind
- Two Pair
- One Pair
- High Card

## How to Run
1. Download the Python file
2. Make sure Python is installed
3. Run:

python poker_game.py
