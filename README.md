# Budget Tracker Python Project

## Description
This is a Python-based Budget Tracker that allows users to:
- Add expenses
- View expenses
- Delete expenses
- View category summary
- Save and load data

## Features
- Uses Object-Oriented Programming
- Includes error handling
- Uses lists, dictionaries, and functions

## How to Run
1. Download the file
2. Run: python budget_tracker.py
